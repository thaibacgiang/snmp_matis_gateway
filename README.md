# SNMP MATIS Gateway for Home Assistant

Tự động phát hiện Attomat MATIS, PIN Lithium , SDM220, DDS,...
Không cần YAML. Tự tạo sensor + switch.

## Cài đặt
Copy thư mục `snmp_matis_gateway` vào `/config/custom_components`.

Hoặc dùng HACS → Custom Repositories → URL repo này → Category: Integration.
